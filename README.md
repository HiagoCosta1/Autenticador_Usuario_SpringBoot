# Securing a Web Application   

Projeto de estudo baseado no guia oficial da Spring:

**Securing a Web Application**
https://spring.io/guides/gs/securing-web

## Tecnologias utilizadas
- Java 17
- Spring Boot
- Spring Security
- Thymeleaf
- Maven

## Objetivo
Demonstrar a implementação de autenticação básica em uma aplicação web utilizando Spring Security, com páginas HTML renderizadas via Thymeleaf.

## Funcionalidades
- Página inicial pública (`/` e `/home`)
- Página protegida (`/hello`)
- Login customizado (`/login`)
- Logout (`/logout`)
- Usuário em memória (InMemoryUserDetailsManager)
